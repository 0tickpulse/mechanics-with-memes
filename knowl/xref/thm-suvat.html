Theorem 2.1.14. Kinematic Equations for Constant Acceleration in One Dimension.

For an object moving in one dimension with constant acceleration \(a\text{,}\) initial velocity \(v_0\text{,}\) and initial position \(x_0\text{,}\) the following equations hold:
\begin{align*} v \amp = v_0 + at\\ \Delta x \amp = v_0 t + \frac{1}{2} at^2\\ v^2 \amp = v_0^2 + 2a\Delta x\\ \Delta x \amp = \frac{v + v_0}{2} t \end{align*}
🔗
🔗

Proof.

The first equation can be derived by integrating the acceleration:
\begin{equation*} v(t) = v_0 + \int_0^t a \, \dd{t'} = v_0 + at\text{.} \end{equation*}
The second equation can be derived by integrating the velocity:
\begin{equation*} x(t) = x_0 + \int_0^t v(t') \, \dd{t'} = x_0 + \int_0^t (v_0 + at') \, \dd{t'} = x_0 + v_0 t + \frac{1}{2} at^2\text{.} \end{equation*}
The third equation can be derived by eliminating time from the first two equations. From the first equation, we have \(t = \frac{v - v_0}{a}\text{.}\) Substituting this into the second equation gives us
\begin{equation*} \Delta x = v_0 \qty( \frac{v - v_0}{a} ) + \frac{1}{2} a \qty( \frac{v - v_0}{a} )^2\text{.} \end{equation*}
Simplifying this expression yields the third equation:
\begin{equation*} v^2 = v_0^2 + 2a\Delta x\text{.} \end{equation*}
The fourth equation can be derived by eliminating acceleration from the first two equations. From the first equation, we have \(a = \frac{v - v_0}{t}\text{.}\) Substituting this into the second equation gives us
\begin{equation*} \Delta x = v_0 t + \frac{1}{2} \qty( \frac{v - v_0}{t} ) t^2\text{.} \end{equation*}
Simplifying this expression yields the fourth equation:
\begin{equation*} \Delta x = \frac{v + v_0}{2} t\text{.} \end{equation*}
